# Circus Board Game

A simple digital board game built with Unity. Up to 4 players take turns rolling a die and moving around a circus-themed board, landing on special tiles that can move them forward, send them back, or make them lose a turn.

---

## Overview

This project is a turn-based board game made in Unity. Players move tokens along a fixed path of tiles; tiles can have a special effect.

---

## Features

- Local multiplayer (hot-seat) board game.
- Different tile types.
- Dice–based movement.
- Character selection.
- Pause menu with resume / quit.
- Basic audio feedback via `SoundEffectsScript`.
- Leaderboard and persistent save data.

---

## How to Download & Run

### Download Release Build (Recommended)

1. Go to the GitHub repository:  
   <https://github.com/DJJ006/BoardGame>
2. Navigate to the **Releases** section (right sidebar or `Releases` tab).
3. Download the latest build for your platform (e.g. `Windows.zip`).
4. Extract the archive to any folder.
5. Run the executable (e.g. `BoardGame.exe`).

---

## How to Play

### Game Setup

1. Launch the game.
2. On the **Main Menu**, you can typically:
- Start a new game.
- View leaderboard (via `MainMenuLeaderboardUI`).
- Open settings (`SettingsScript`).
3. On the **Character Select** screen (`CharacterSelectScript`), choose a character/token for each player if available.
4. Proceed to the **Game Scene**.
